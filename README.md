# 가계부 스켈레톤 프로젝트

## ✅ commit 규칙
- message: type: 제목
   - feat: 새로운 기능 추가
   - fix: 버그 수정
   - docs: 문서 내용 변경
   - style: 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 등
   - refactor: 코드 리팩토링
   - test: 테스트 코드 작성
   - chore: 빌드 수정, 패키지 매니저 설정, 운영 코드 변경이 없는 경우 등
- description: 자유
```
git commit -m "Feat: 회원 가입 기능 추가"
```

## ✅ PR 규칙
- 제목: [#기능 번호] 변경 사항   
  <code>[#1] 회원가입 기능 구현</code>
- comment는 자유
